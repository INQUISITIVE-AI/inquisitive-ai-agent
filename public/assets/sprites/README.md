# Pixel Office Sprite Assets

## How to get the exact Pixel Agents look

The visual quality comes from professional pixel art tilesets. Buy ONE of these:

### Option A: Donarg's Office Interior Tileset ($2) — EXACT match to Pixel Agents
1. Go to: https://donarg.itch.io/officetileset
2. Buy for $2
3. Download the ZIP
4. Extract and copy these files into THIS folder (`public/assets/sprites/`):
   - `tileset-16x16.png` → rename to `furniture.png`
   - The character sprites → rename to `characters.png`

### Option B: LimeZu's Modern Office ($4.95) — even more furniture options
1. Go to: https://limezu.itch.io/modernoffice
2. Buy for $4.95
3. Download the ZIP
4. From the `16x16/` folder, copy:
   - `Room_Builder_Office_16x16.png` → rename to `tiles.png`
   - `office_furniture_16x16.png` → rename to `furniture.png`
   - Character sheets → rename to `characters.png`

After placing the PNGs here, the office will automatically load them.
Without these files, the office uses built-in procedural sprites as fallback.
